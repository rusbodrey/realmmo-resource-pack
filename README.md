Публичное хранилище официального ресурс-пака Minecraft-сервера RealMMO. Серверные исходники и конфигурация здесь не публикуются.

SHA-1 `RealMMO-Resources-26.1.2-v7.zip`: `d82724d107d1818e09efd97091f8ec1bb5befeb0`

Версия v7 исправляет цельные сегментированные кнопки аукциона, добавляет управление ценой от нуля, явный выбор лота и компактный баланс в заголовке. Косметическая локализация offline-чата и валютный HUD сохранены.
