# RealMMO Resource Pack

Публичное хранилище официального ресурс-пака Minecraft-сервера RealMMO. Серверные исходники и конфигурация здесь не публикуются.

SHA-1 `RealMMO-Resources-26.1.2-v14.zip`: `83955bc7271ce865387a7c258f17ead283005a96`

Версия v14 использует предоставленное высокоразрешённое изображение кнопок: каждая кнопка физически разрезана на три отдельных квадратных глифа 256×256.
